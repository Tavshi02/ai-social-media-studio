# 🚀 AI Social Media Studio

An AI-powered web application that converts simple ideas into high-quality, structured social media carousel content.

---

## ✨ Features

* 🎯 Convert any topic into engaging carousel slides
* 🧠 Structured content generation:

  * Hook
  * Problem
  * Insight
  * Example
  * Call To Action
* 🚫 Smart input validation (rejects garbage input)
* 🎨 Clean and modern UI
* 📱 Responsive design
* ⚡ Real-time content generation via backend API

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Deployment:**

  * Frontend → Netlify
  * Backend → Render
* **Version Control:** Git & GitHub

## ⚙️ How It Works

1. User enters a topic
2. Frontend sends request to backend
3. Backend validates input
4. Structured content is generated
5. Slides are displayed in carousel format
